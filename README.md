<h1 align="center">F1 Data API</h1>

Simple API all about Formula 1 data.

### Install and run
```
# Download API data script from Ergast website and insert the data into the MySQL database
http://ergast.com/mrd/db/

# Clone the repository
https://github.com/jdro10/f1-data-api
cd f1-data-api

# Install dependencies
npm install

# Start the project
npm start

# Open SWAGGER UI for API documentation and test the API
http://localhost:3000/docs/
```

### API data

* [Ergast API](http://ergast.com/mrd/) - Formula One data.
